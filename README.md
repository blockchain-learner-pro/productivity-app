# Offline Productivity App

A lightweight offline-first productivity app designed to help users manage tasks and notes seamlessly, even without an internet connection.

## 🚀 Live Demo

https://jovial-genie-3a6596.netlify.app/

## ✨ Features

* Works offline (PWA enabled)
* Runs in the browser
* Installable on mobile
* Simple and fast interface

## 📸 Screenshots

![App Screenshot](./screenshots/app.png)

## 🛠 Tech Stack

* React
* Service Workers (PWA)
* Netlify (deployment)

## ▶️ How to Run Locally

```bash
npm install
npm start
```

## 📦 Build for Production

```bash
npm run build
```
